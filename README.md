# MiniTranslator

A simple, lightweight desktop translator built with Python and PySide6. Fast, easy to use, and sits quietly in your system tray until you need it.

## Features
- **Instant Translation** – Select text and press `Ctrl+Alt+Shift+Q` (default) to translate immediately.
- **Modern UI** – Smooth animations and a clean interface.
- **Themes** – Dark, Light, and Lavender modes.
- **History** – Keeps your recent translations for quick access.
- **Japanese Support** – Romaji conversion included for Japanese text.
- **Autostart** – Option to start automatically with Windows.

##  Installation

### For Users
1. Go to the [Releases](https://github.com/hellattrigger/MiniTranslator/releases) page.
2. Download `MiniTranslator.exe`.
3. Run it.
