MiniTranslator

	A minimalist system tray translator. Select any text and translate it instantly using hotkeys.

	Tray-based: Can be minimized to the system tray to stay out of your way.

	Customizable Hotkeys: Easily change the translation shortcut in settings.

	Japanese Support: Includes Romaji for Japanese text translations.

	Multilingual: Switch both the interface language and the translation direction.

	Themes: Choose from 3 different visual themes.
 Installation

 For Users
1. Go to the [Releases](https://github.com/hellattrigger/MiniTranslator/releases) page.
2. Download `MiniTranslator.exe`.
3. Run it and start translating!

 For Developers
1. Clone the repository:
   ```bash
   git clone [https://github.com/hellattrigger/MiniTranslator.git](https://github.com/hellattrigger/MiniTranslator.git)
   Install dependencies: pip install PySide6
   Run the application: python app.py
