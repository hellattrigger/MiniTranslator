#MiniTranslator

A lightweight, stylish desktop translator built with Python and PySide6. Fast, intuitive, and stays out of your way in the system tray.

## Features
- Instant Translation: Use `Ctrl+Alt+Shift+Q` (default) to translate selected text.
- Modern UI: Smooth animations and a clean, user-friendly interface.
- Theming: Support for Dark, Light, and Lavender themes.
- History: Keep track of your previous translations.
- Japanese Support: Built-in Romaji conversion for Japanese text.
- Autostart: Option to launch automatically with Windows.

##  Installation

### For Users
1. Go to the [Releases](https://github.com/hellattrigger/MiniTranslator/releases) page.
2. Download `MiniTranslator.exe`.
3. Run it and start translating!
